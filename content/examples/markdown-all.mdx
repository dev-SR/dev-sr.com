For remote images, you can also use the `remotePatterns` option in `next.config.js` to allow remote images to be loaded.

Example:

```js
const nextConfig = {
  images: {
    remotePatterns: [
      {
        protocol: 'https',
        hostname: 'images.unsplash.com',
      },
    ],
  },
};
export default nextConfig;
```
